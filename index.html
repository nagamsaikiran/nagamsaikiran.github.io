About Me

I’m a Senior Full Stack .NET Developer with 9+ years of experience building secure, scalable, and performance-driven enterprise applications across finance, healthcare, and banking domains.

I specialize in designing end-to-end solutions using .NET 8, ASP.NET Core, Angular, React, Blazor, C#, TypeScript, SQL Server, and Azure Cloud. My work spans from architecting backend APIs and microservices to developing modern, responsive front-end applications with optimized performance and clean component design.

Over the years, I’ve led development of cloud-native systems, implemented secure authentication mechanisms (OAuth2, JWT), optimized database performance, containerized applications using Docker, and built CI/CD pipelines with Azure DevOps to ensure reliable production releases.

Beyond coding, I focus on:

  • Writing clean, maintainable, and testable code
  • Improving performance and scalability
  • Applying design patterns and SOLID principles
  • Delivering business-critical features in Agile environments

I believe great software is not just about working code — it’s about reliability, clarity, scalability, and long-term maintainability.
