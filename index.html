About Me

I am a Full Stack Developer and Software Developer who enjoys building useful products end to end, from backend services and databases to modern front-end experiences and cloud deployment.

My core strength is full-stack engineering with .NET, ASP.NET Core, Angular, React, TypeScript, SQL Server, PostgreSQL, and Azure. I also like exploring AI-powered features, including document chat, retrieval-augmented generation, and practical workflows that make applications more useful for real users.

I do not want to limit myself to only one title or one stack. I am focused on becoming the kind of developer who can understand a product problem, design a maintainable solution, build the right features, and keep improving the system over time.

What I focus on:

  • Building complete full-stack features from API to UI
  • Designing secure, scalable, and maintainable systems
  • Adding AI features where they create real product value
  • Learning beyond one framework so I can choose the right tool for the work

My goal is to keep growing as a developer who can contribute across the full software lifecycle: architecture, implementation, deployment, performance, and user experience.
